Layout Patterns

The compositional skeletons every section in the library is built on. Learn these five and you can assemble new sections without writing new grid CSS.

.layout-split

Two columns, ratio set by --split-ratio-a/b/wide or a custom --split. Add [data-flip="true"] to mirror column order.
